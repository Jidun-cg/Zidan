Mode Tutorial : Lv. 1 Pemula Frontend

HTML (Map/Stage)

HTML itu seperti peta level dalam game. HTML menentukan letak objek: platform, pintu, musuh, dan item. Tanpa peta, nggak ada apa apa cuma ruang kosong

CSS (Tampilan/Skin)

CSS itu skin, warna, dan efek partikel. CSS bikin semuanya terlihat keren, mulai dari tekstur, ukuran, posisi halus, dan animasi kecil. HTML memberi struktur, CSS memberi tampilan

JavaScript (Logika)

JS menambah interaksi pada website kita seperti toggle menu, efek ketik, scroll reveal, dan suara bip saat hover

Tailwind (Utility CSS)

Tailwind itu seperti inventory barang yang berisi banyak item siap pakai (utility), dengan Tailwind kita dapat dengan cepat memasang warna, margin, atau tombol tanpa membuat skin dari nol. Jadi kita bisa bangun visual cepat tanpa menulis banyak kode CSS
