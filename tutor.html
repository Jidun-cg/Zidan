Tutorial Membuat Website Sederhana

Tujuan: Membuat halaman sederhana yang menampilkan teks “Hello World” lalu berubah saat tombol diklik

1) Siapkan Tiga File

  • index.html
  • style.css
  • script.js

2) Isi index.html

<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Hello World</title>
  <link rel="stylesheet" href="style.css">
  <script src="script.js" defer></script>
</head>
<body>
  <h1 id="title">Hello World</h1>
  <button id="btn">Ubah Teks</button>
</body>
</html>

3) Isi style.css

html, body { height: 100%; }
body{
  font-family: 'Press Start 2P', cursive, Arial, sans-serif;
  background: #0e0e0e; color: #fff;
  display:flex; align-items:center; justify-content:center;
  flex-direction:column; gap:16px;
}
h1{ font-size: 18px; }
button{
  background: #ffd166; color:#000;
  border:4px solid #000; outline:4px solid #ffd166;
  padding:10px 12px; cursor:pointer;
  box-shadow: 0 4px 0 #000, 0 8px 0 #ffd166;
}
button:active{ transform: translateY(2px); box-shadow: 0 2px 0 #000, 0 4px 0 #ffd166; }

4) Isi script.js

document.addEventListener('DOMContentLoaded', () => {
  const title = document.getElementById('title');
  const btn = document.getElementById('btn');
  btn.addEventListener('click', () => {
    title.textContent = 'Hello World (Clicked)!';
  });
});

Jika ketiga file berada dalam folder yang sama, buka file index.html di browser. Kamu akan melihat teks “Hello World” dan tombol “Ubah Teks”. Klik tombolnya untuk melihat perubahan.
